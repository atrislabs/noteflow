## Description

**Write beautifully. Store locally. Sync intelligently.**

Noteflow is a native macOS markdown editor that renders your writing beautifully as you type. No split panes. No preview mode. Just write and watch your markdown transform into styled text in real-time.

**WYSIWYG Markdown Editing**
- Headers, bold, italic, and code render instantly
- Clickable links that actually work
- Task lists with toggleable checkboxes
- Code blocks with syntax styling
- Blockquotes, highlights, and strikethrough

**Your Files, Your Way**
- Notes stored as plain .md files on your Mac
- Choose any folder as your notes directory
- Full portability - no proprietary format
- Works offline, no account required

**Keyboard-First Design**
- Notion-style shortcuts (⌘B, ⌘I, ⌘K)
- Slash commands for quick formatting
- Quick Open (⌘P) to jump to any note
- Tab to indent lists, Enter to continue

**Stay Organized**
- Folder hierarchy mirrors your filesystem
- Tag notes with #hashtags
- Mark favorites for quick access
- Full-text search across all notes

**Atris AI Integration (Optional)**
- Sync notes to your Atris AI agent
- Make your knowledge accessible to AI
- One-click cloud backup
- Completely optional - works offline

**Privacy First**
- Notes never leave your Mac unless you sync
- No analytics, no tracking
- No account required for local use

## Review Notes for Apple

Noteflow is a markdown note-taking app that stores notes as local files. The app includes optional cloud sync to Atris AI (https://atris.ai), which requires an Atris account and API token. This feature is entirely optional and the app works fully offline without it.

The app requests:
- File access: To read/write markdown notes in user-selected folder
- Network access: Only used for optional Atris sync feature

No account is required to use the app locally.
