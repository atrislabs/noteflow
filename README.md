# Noteflow

A beautiful, native markdown editor for macOS. Write beautifully. Store locally. Sync intelligently.

![Noteflow](https://atris.ai/noteflow/og-image.png)

## Features

### WYSIWYG Markdown
Real-time markdown rendering as you type. No preview pane needed.

- **Bold**, *italic*, `code`, ~~strikethrough~~, ==highlights==
- Headers (H1-H6) with proper typography
- Bullet, numbered, and task lists with auto-continuation
- Code blocks with syntax highlighting
- Blockquotes, links, and images
- Tables with monospace alignment

### Local-First Storage
Your notes are just `.md` files on your filesystem. No vendor lock-in.

- Works with any folder structure
- Compatible with Obsidian, iA Writer, and other markdown apps
- Full control over your data

### Focus Features
- **Focus Mode** (⌘⇧F) - Distraction-free writing
- **Typewriter Mode** (⌘⇧T) - Keep cursor centered
- **Document Outline** (⌘\\) - Navigate by headers

### Smart Writing
- **Slash Commands** - Type `/` for quick formatting
- **Daily Notes** (⌘D) - Quick journal entries
- **Templates** (⌘⌥N) - Meeting notes, projects, brainstorms, and more
- **Quick Open** (⌘P) - Fast note switching

### Atris Sync
Optional cloud sync with [Atris AI](https://atris.ai) for backup and AI-powered features.

| ⌘⇧T | Typewriter Mode |

| ⌘B | Bold |
| ⌘I | Italic |
| ⌘K | Insert Link |
| ⌘1-3 | Headings |

## Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon or Intel Mac

## Installation

Download the latest release from [atris.ai/noteflow](https://atris.ai/noteflow) or the Mac App Store.

## Building from Source

1. Clone the repository
2. Open `Scribe.xcodeproj` in Xcode 15+
3. Build and run (⌘R)
